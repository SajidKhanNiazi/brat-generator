# Image Placeholders

This directory contains placeholder image files that should be replaced with proper branded images.

## Files to Replace

### `/public/og-image.png`
- **Size**: 1200 × 630 pixels
- **Purpose**: Social media preview image (Open Graph)
- **Content**: Should include your logo, app name "Brat Generator", and branding
- **Background**: Consider using the signature green color (#8ACE00) from your brand
- **Format**: PNG (or high-quality JPG)

### `/public/favicon.ico`
- **Size**: 16×16, 32×32, or 48×48 pixels (multi-size ICO recommended)
- **Purpose**: Browser tab icon, bookmarks
- **Content**: Simple icon representing the Brat Generator (could be a sparkle icon, "B" letter, or similar)
- **Format**: ICO file format

## How to Create These Images

1. **og-image.png**: Use design tools like:
   - Figma, Canva, Photoshop, or similar
   - Include text: "Brat Generator" with your branding
   - Use colors: #8ACE00 (green) and dark background

2. **favicon.ico**: 
   - Create a square icon (32×32 or 48×48)
   - Convert to ICO format using online tools or image editors
   - Or use a favicon generator service

The current files are minimal placeholders and should be replaced before production deployment.
